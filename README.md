# 📊 Customer Behaviour Analysis

## Overview

Customer Behaviour Analysis is an end-to-end Data Analytics project that examines customer purchasing patterns, subscription behavior, product preferences, and spending trends.

The project demonstrates the complete analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive dashboard development using Power BI.

The objective is to generate actionable business insights that can help improve customer engagement, increase revenue, and support data-driven decision-making.

---

## Dataset

The dataset contains customer shopping behavior information, including:

- Customer Demographics
- Gender
- Age Group
- Product Categories
- Purchase Amounts
- Subscription Status
- Shipping Preferences
- Customer Ratings

## Key Results & Insights

### Customer Base
- Analyzed approximately 3,900 customers.

### Subscription Behavior
- Around 73% of customers are non-subscribers.
- Only 27% are subscribed customers.

### Category Performance
- Clothing generates the highest revenue.
- Accessories are the second-best performing category.
- Outerwear contributes the lowest revenue.

### Customer Spending
- Average purchase amount is approximately $59.76.

### Customer Satisfaction
- Average customer rating is 3.75/5.

### Age Group Analysis
- Young Adult and Middle-Aged customers contribute the highest revenue and sales.
